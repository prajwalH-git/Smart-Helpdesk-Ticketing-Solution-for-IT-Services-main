# Smart-Helpdesk-Ticketing-Solution-for-IT-Services

An AI-based IT helpdesk system that categorizes, assigns, and resolves support tickets automatically.

## OpenAI setup

Keep your API key on the backend in `Backend/.env`. Do not place it in the React frontend.

Required variables:

- `OPENAI_API_KEY`
- `OPENAI_MODEL=gpt-4o-mini`
- `MONGO_URI`
- `JWT_SECRET`
